## Lazyload
一个javascript图片懒加载库

## Usage
        <div class="x x1">
            <img data-src="imgs/family_account.png" />
        </div>
        <div class="x x2">
            <img data-src="imgs/framework.png" />
        </div>
        <div class="x x3">
            <img data-src="imgs/manager.jpg" />
        </div>
        <script src="lazyload.js"></script>
        <script>
            new LazyLoad('.x');   
        </script>
